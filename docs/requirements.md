# 项目需求文档

## 项目概述

**项目名称**: HN每日新闻简报系统  
**项目代号**: 歪朝会议第一次  
**目标用户**: 程序员

## 核心需求

### 功能需求

1. **RSS自动抓取**
   - 每天北京时间8:00自动执行
   - 支持60+个RSS源（来自HN热门博客OPML）
   - 去重处理（避免重复抓取）

2. **内容处理**
   - 提取文章正文内容
   - AI生成100-200字中文摘要
   - 保留原始链接

3. **内容展示**
   - Markdown格式存储
   - 响应式Web界面
   - 历史新闻浏览
   - 暗色模式支持

4. **自动部署**
   - GitHub Pages托管
   - CI/CD自动化流程

### 非功能需求

1. **性能**
   - 页面加载时间 < 3秒
   - RSS抓取超时 < 10秒/源

2. **可用性**
   - 99%+ 可用性
   - 自动失败重试

3. **成本**
   - 月运营成本 < $20
   - 优先使用免费服务

## 技术栈

### 前端
- React 18+
- Vite
- Tailwind CSS
- React Router

### 后端
- Python 3.11+
- FastAPI
- feedparser
- OpenAI API

### 基础设施
- GitHub Actions
- GitHub Pages
