# 需求清单

**项目代号：** 歪朝会议第一次
**创建时间：** 2026-03-11
**目标用户：** 程序员每日新闻热点推送

---

## 📌 项目概述

开发一个RSS订阅聚合系统，每天北京时间8点自动抓取Hacker News相关博客内容，通过AI生成100-200字简报，推送到GitHub Pages供程序员阅读。

### 技术栈
- **前端：** React（基于模板 GitHub - 782042369/react-template）
- **后端：** Python
- **部署：** GitHub（统一仓库）
- **展示：** GitHub Pages

---

## 🎯 核心功能需求

### 1. RSS抓取服务（后端）
- [ ] 从OPML源解析RSS订阅列表
  - 源地址：https://gist.github.com/emschwartz/e6d2bf860ccc367fe37ff953ba6de66b
- [ ] 每天北京时间8点定时执行
- [ ] 支持并发抓取多个RSS源
- [ ] 错误重试机制（3次）
- [ ] 去重处理（基于URL或标题）

### 2. AI内容总结（后端）
- [ ] 为每条RSS条目生成100-200字摘要
- [ ] 保留原始标题和链接
- [ ] 支持批量处理
- [ ] API调用失败降级方案（返回原文前200字）

### 3. 数据管理（后端）
- [ ] 数据存储方案（SQLite / JSON文件）
- [ ] 历史记录归档（按日期分文件夹）
- [ ] 缓存机制（避免重复抓取）
- [ ] 数据结构设计

### 4. 前端展示（React）
- [ ] 基于react-template快速搭建
- [ ] MD文件渲染（支持Markdown）
- [ ] 响应式布局（移动端友好）
- [ ] 按日期分类浏览
- [ ] 搜索功能（标题/内容）
- [ ] 暗色模式支持

### 5. 自动化部署
- [ ] GitHub Actions配置
  - 每天8:00（UTC+8）触发后端抓取
  - 自动生成MD文件
  - 自动推送到仓库
- [ ] 前端自动构建部署
